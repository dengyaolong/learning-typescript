# learning-typescript
学习typescript的demo

## typescript简介
typescript是javascript的超集,顾名思义,typescript = type + javascript. 是一种有强类型的脚本语言.

其优点包含但不仅限于:

* 可读性
* 可维护性
* 活跃的社区　

缺点：

* 学习成本

## 安装typescript

```
npm install -g typescript
```
